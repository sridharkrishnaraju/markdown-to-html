# Markdown to HTML Converter

A Markdown to HTML converter: paste Markdown, get clean semantic HTML plus a live preview, all in the browser. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="markdown-tool.js"></script>
<markdown-tool></markdown-tool>
```

## What it does

Markdown is the lightweight syntax behind README files, notes and docs. Browsers and most CMS fields need HTML, though. This converter bridges that gap — headings, bold/italic, code, links, images, lists, blockquotes and rules become real semantic tags.

## When you'd use it

Moving notes or a README into a CMS, or turning Markdown drafts into publishable HTML.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
