# Markdown to HTML Converter

A Markdown to HTML converter: paste Markdown, get clean semantic HTML plus a live preview, all in the browser. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/markdown-to-html)**

```html
<script src="markdown-tool.js"></script>
<markdown-tool></markdown-tool>
```

## What it does

Markdown is the lightweight syntax behind README files, notes and docs. Browsers and most CMS fields need HTML, though. This converter bridges that gap — headings, bold/italic, code, links, images, lists, blockquotes and rules become real semantic tags.

## Install

```bash
npm install @sgbp/markdown-tool
```

or copy `markdown-tool.js` into your project.
